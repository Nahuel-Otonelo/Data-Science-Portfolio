# Data Science Portfolio

Repositorio con una colección de mis proyectos de Data Science, abarcando desde análisis exploratorio y tests A/B hasta MLOps y NLP.

## Proyectos
*   **01_Test_AB_pricing**: Clústering y Test A/B.
*   **02_Next_Product_Prediction**: Modelos de clasificación.
*   **03_Machine_Learning_Engineer_Exam**: Pruebas técnicas de MLE.
*   **04_Proyecto_MLOps**: Despliegue y orquestación de modelos.
*   **05_Proyectos_NLP**: Modelos de lenguaje y traducción.
